# Jarvis AI Assistant

## Overview

Jarvis is a personal AI assistant that can perform various tasks like sending emails, fetching news, translating text, reading PDF files, and more. It utilizes various libraries and APIs to provide a wide range of functionalities.

## Features

- **Text-to-Speech**: Uses `pyttsx3` for speaking.
- **Speech Recognition**: Recognizes commands using `speech_recognition`.
- **Email Sending**: Sends emails with optional attachments.
- **News Fetching**: Retrieves news headlines.
- **Translation**: Translates text into different languages.
- **PDF Reading**: Extracts text from PDF files.
- **Camera Operations**: Opens and closes the webcam.
- **Graphical User Interface**: Built using PyQt5.


## Libraries Used

- **pyttsx3**: A text-to-speech conversion library in Python.
- **speech_recognition**: Recognizes speech using various speech recognition APIs.
- **datetime**: Provides classes for manipulating dates and times.
- **os**: Provides a way of using operating system-dependent functionality.
- **cv2**: OpenCV library for computer vision tasks.
- **random**: Implements pseudo-random number generators.
- **requests**: Allows sending HTTP requests.
- **wikipedia**: Fetches information from Wikipedia.
- **webbrowser**: Provides a high-level interface to allow displaying Web-based documents to users.
- **pywhatkit**: Performs various tasks like sending WhatsApp messages, playing YouTube videos, etc.
- **smtplib**: Defines an SMTP client session object that can be used to send mail.
- **sys**: Provides access to some variables used or maintained by the Python interpreter.
- **pyjokes**: Allows you to get programming jokes for fun.
- **pyautogui**: Provides functions for controlling the mouse and keyboard.
- **time**: Provides various time-related functions.
- **instaloader**: Downloads pictures (or videos) along with their captions and other metadata from Instagram.
- **googletrans**: Library to translate text using Google Translate.
- **fitz**: Part of PyMuPDF, used for reading and manipulating PDF files.
- **psutil**: Provides an interface for retrieving information on running processes and system utilization.
- **PyQt5**: Set of Python bindings for Qt libraries which can be used to create cross-platform applications.

## Troubleshooting

1. **Dependencies**: Ensure all dependencies are installed and up to date.
2. **Microphone Access**: Ensure your microphone is working and accessible by the script.
3. **Environment Variables**: Double-check that your environment variables for email are correctly set.
4. **Error Handling**: The script contains basic error handling. If you encounter an issue, the error message printed in the console will help diagnose the problem.
